Bomberman
===========

Description
-----------
Simple game for 2 or even 3 players (offline) written in C using Allegro 4.2 libary. 

Installation
------------

### Run

    make
    ./Bomberman

To build on linux first install liballegro4.2-dev. Repo contains required libaries for Windows.
